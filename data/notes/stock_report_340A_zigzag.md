## 2. 事業内容・サービス詳細

### 2-1. ビジネスモデル概要

ジグザグは「**タグ1行で越境ECをまるごとサポート**」を強みとする越境EC支援プラットフォーム。
国内ECサイト運営者が自社サイトにJavaScriptタグを1行埋め込むだけで、最短1日で228の国・地域への越境販売が可能になる。

```
【収益構造の概要】

国内ECサイト（加盟店）
      ↓ JSタグ1行設置
  ジグザグのシステム
      ↓
  海外カスタマー
      ↓
  ジグザグが代行：多言語対応・決済・通関・海外配送・カスタマーサポート
      ↓
  売上の大半（95%）：海外カスタマーからの購入代行手数料
```

### 2-2. 主要サービス

#### ① WorldShopping BIZ（国内EC事業者向け）

| 項目 | 内容 |
|------|------|
| 対象 | 国内ECサイト運営者 |
| 導入方法 | JavaScriptタグ1行の追加のみ |
| 対応国・地域 | **228の国・地域** |
| 初期費用 | 3万円 |
| 月額費用 | 5,000円 |
| 販売手数料 | **0円**（事業者側への請求なし） |
| 代行内容 | 多言語カスタマーサポート・国際決済・海外配送・通関手続き |
| 対応決済 | Alipay・PayPal・銀聯・WeChat Pay・Amazon Pay等 |
| 特許機能 | 除外設定機能（国・地域・ブランド単位での販売制御） |

**特徴：**
- 既存ECサイトの改修不要。業務フローの変更なく越境販売が可能
- 数千店舗以上の導入実績によるデータ蓄積（アクセス・注文データの横断活用）
- futureshop・ecbeingなど主要ECプラットフォームと業務提携済み

#### ② WorldShopping（海外カスタマー向け）

海外在住の消費者向けに、日本のECサイトから商品を購入・転送する購入代行サービス。
多言語サポート・多様な決済・海外配送をフルカバー。出荷実績は172の国・地域。

#### ③ インバウンド支援モデル（2025年〜新展開）

訪日外国人向けに「**旅マエ・旅ナカ・旅アト**」の3ステージで一貫支援する新サービス。
インバウンド需要の取り込みを越境EC支援と組み合わせ、デジタルとリアルを融合させた導線設計を提供。

### ② 為替リスク

| リスク内容 | 影響 |

| 為替変動全般 | 売上高の95%が海外依存のため影響大 |

### ③ 米国関税・国際規制リスク

2025年後半から米国の関税強化が進行。越境EC事業者への影響：
- 米国向け越境ECの需要減退リスク
- 通関コストの上昇
- 会社側は「関税前払方式」を導入して対応中

### ④ 競合・代替サービスの登場

- 日本でのBTC-ETF解禁（※BTC関連ではなく）→ 日本でのBuyee型サービスの競合増加
- Shopify Marketsなどの自社対応ツールの普及による「タグ代行不要論」の台頭
- 競合：BEENOS（tenso/Buyee）、Shopify、Amazon Global等

### ⑤ 上場間もない小型株リスク

- 2025年3月上場。市場での評価形成が途上
- 東証グロース小型株：流動性リスク（大量売買時のスプレッド拡大）
