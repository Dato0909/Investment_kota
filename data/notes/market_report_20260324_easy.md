## 1. 「円」と「ドル」って何が変わったの？

### まず「為替」ってなに？

> たとえば、1ドル＝158円 というのは「アメリカのお金1枚と、日本のお金158枚が同じ価値」ということ。
> これが毎日少しずつ変わるんだ。

### 3月24日のドル円の動き

```
    円安（ドル高）方向
         ↑
 159.18円 ★ 今日の最高値
         |
 158.73円 ◆ 1日の終わり値（クローズ）
         |
 158.40円 ● 朝の値（スタート）
         |
 158.30円 ▼ 今日の最安値
         ↓
    円高（ドル安）方向
```

**なぜ動いたの？**

| 時間 | 出来事 | 円はどうなった？ |
|------|--------|----------------|
| 朝〜昼 | 「中東で戦争が終わりそう」とトランプ大統領が発言 | 円が少し安くなった（ドル高） |
| 昼〜夕 | 日本の財務大臣が「円安が続いたら対応する」と発言 | 上がりにくくなった |
| 夕〜夜 | 「実は停戦はまだ決まっていない」という報道 | 一気に円高に戻った |

### 円安・円高って何がいいの？悪いの？

```
【円安（1ドル＝160円に近づくとき）】
  ✅ 日本の会社が外国でものを売りやすい（トヨタなどが儲かる）
  ✅ 外国人観光客が日本に来やすくなる
  ❌ 輸入品（食料・エネルギー）が高くなる
  ❌ 海外旅行が高くなる

【円高（1ドル＝140円に近づくとき）】
  ✅ 輸入品が安くなる
  ✅ 海外旅行が安くなる
  ❌ 日本の会社の輸出が不利になる
```

- 安心すると、みんな株を買おうとするので値段が上がる

## 3. 石油（原油）の価格が急落！なんで？

### 原油ってなに？なんで大事なの？

> 車のガソリン、プラスチック、電気など、私たちの生活のほぼすべてに原油が使われている。
> 原油が高くなると → ガソリン代・電気代・食品の値段が全部上がる。

### 今日の原油の大ドラマ

```
  今日の原油価格の動き：

  92ドル ──────── 朝〜夕方（中東緊張で高止まり）
           \
            \
  86ドル ──── ここまで急落！（「停戦交渉中」の報道が出た瞬間）
            /
           /
  92ドル ── 最終的にここで引け

  ※わずか数時間で92→86→92ドルと激しく動いた！
```

**なぜ原油と中東は関係あるの？**

> 中東（サウジアラビア・イランなど）は世界の石油の約3割を生産している地域。
> そこで戦争が起きると「石油が足りなくなるかも！」と思った人がたくさん買うから値段が上がる。
> 逆に「戦争が終わりそう」になると値段が下がる。

## 4. 「金利」って何？なんで株に関係するの？

### 金利をわかりやすく説明すると…

> **「お金を借りるときの料金（レンタル料）」** のこと。
>
> 例：銀行から100万円を借りて、金利が年5%なら、1年後に105万円返す。

### アメリカの金利が上がった

今日、アメリカの10年国債の金利が **4.37%** に上昇した。

```
  金利が上がる → 債券（こくさい）を買う方が安全で儲かる
             → 株を売って債券を買う人が増える
             → 株の値段が下がりやすい

  でも今日は「中東停戦期待」の方が強かったので株は上がった！
```

## 6. 「サポート」と「レジスタンス」って何？

テクニカル分析（チャート分析）の基本用語を中学生向けに！

```
  【レジスタンス（上値抵抗）】
  → 「天井」みたいなもの。価格がここまで来ると「高すぎる！」と
    売る人が増えて、それ以上上がりにくくなる。

  【サポート（下値支持）】
  → 「床」みたいなもの。価格がここまで来ると「安い！買おう！」と
    買う人が増えて、それ以上下がりにくくなる。

  ドル円の場合（3/24現在）：

  160.00円 ━━━━ 天井（政府が介入しそうな水準）
  159.18円 ─────  最近の高値・超えにくい壁
  ─────────────── ←今ここ（158.73円）
  158.30円 ─────  最近の安値・下がりにくい床
  157.80円 ━━━━ もう一段下の床
```
