## 2. 事業内容・サービス詳細

### 2-1. tenso株式会社（BEENOS主力子会社）のサービス構成

#### ① Buyee（代理購入サービス）

海外消費者向けの日本商品購入代行サービス。
国内ECサイトと商品データを連携し、Buyeeプラットフォーム内に海外向け販売ページを自動構築。

| 機能 | 詳細 |
|------|------|
| 多言語対応 | ○ |
| 決済 | PayPal・Alipay・銀聯・クレジットカード等 |
| 海外発送 | 代行（独自ロジスティクス） |
| カスタマーサポート | 多言語対応 |
| 検品・配送補償 | オプションで提供 |
| 対応サイト | 1,850サイト以上（2023年時点） |

**ジグザグとの違い：**
Buyeeは「Buyeeというプラットフォーム上」に商品ページが展開される。
ジグザグの「自社ECサイトのまま越境対応」とは異なり、ECサイト事業者はBuyeeの集客力を活用できる一方、自社ブランド体験は一部Buyeeに依存する。

#### ② 転送コム（海外転送サービス）

日本のECサイトで購入した商品を、海外消費者に代わって受け取り・転送する。
国内ECサイト側の改修は不要だが、消費者自らが購入手続きを行う必要がある。

#### ③ Buyee Connect（越境EC支援・B2B）

既存ECサイトへの越境購入サポートを組み込む企業向けサービス。
BUYMA（エニグモ）・BuySellTechnologies等への導入実績あり。
**グループ累計6,000件超の越境EC支援実績**（2025年時点）。

#### ④ Logi-link（国際配送サービス・2023年〜）

国際物流のアウトソーシングサービス。
商品の保管・注文処理・発送を代行。1パッケージあたり450円〜という価格設定。

## 5. ジグザグ（340A）との比較・競合分析

### ビジネスモデルの根本的な違い

```
【ジグザグ（340A）のモデル】
  ECサイト事業者が主役 → 既存ECサイトのまま海外販売
  タグ1行で自動越境対応
  海外消費者はECサイト事業者の「自社サイト」で購入体験

【BEENOS/Buyeeのモデル】
  Buyeeプラットフォームが主役 → Buyee上に商品ページを展開
  ECサイト側への直接開発変更は不要だが、Buyeeに集客を依存
  海外消費者は「Buyeeサイト」上で購入体験
```

### 定量比較

| 指標 | ジグザグ（340A） | BEENOS（3328） |
|------|----------------|---------------|
| 売上高（直近通期） | 14億円 | 183億円（大型売却後） |
| 営業利益率 | **22.8%** | **22.9%**（改善後） |
| ROE | **約48%** | 高水準（要確認） |
| 事業規模 | 小（数千店舗） | 大（1,850サイト以上・6,000件超） |
| 上場状況 | 現役（2025年上場） | **MBO進行中（上場廃止見込み）** |
| 成長ステージ | **高成長期** | 構造改革・成熟期 |
| 為替依存 | 95%（海外売上） | 高（Eコマースの主力が越境） |

### 競合関係の本質

**直接競合ではなく「補完的競合」の側面が強い：**

- ジグザグは「自社ECサイト志向の事業者」向け
- BEENOSは「プラットフォーム集客力を活用したい事業者」向け

ただし、Buyee Connectは自社ECサイトへの組み込み型であり、ジグザグと直接的に競合する領域。

**BEENOSのMBOはジグザグにとって中長期的な追い風：**
```
BEENOS上場廃止 → 機関投資家・証券会社の越境EC関連注目が
                 ジグザグ（340A）に集中する可能性
               → 比較分析によりジグザグの優位性が際立つ
               → 株価評価の上昇余地
```
