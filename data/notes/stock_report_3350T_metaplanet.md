## 5. リスク分析（慎重ケース）

### ① ビットコイン価格の直撃リスク

```
BTC価格が下落 → 含み損拡大 → 財務悪化 → 株価急落

過去の例：
・2022年のBTC暴落時（70,000ドル→16,000ドル）では
  MSTRが株価90%超の下落を経験
・同様のリスクがメタプラネットにも存在する
```

### ② mNAVプレミアムの剥落リスク

```
現在：株式時価総額 > BTC保有額（プレミアム状態）

プレミアムが消えると：
・純粋なBTC保有額まで株価が収縮
・BTC価格が横ばいでも株価は大幅下落しうる

プレミアムが消える条件：
- 日本でのビットコインETF解禁（直接投資手段の普及）
- 競合他社の登場（類似戦略を採る他の上場企業）
- 経営陣への信頼失墜
```

### ③ 希薄化リスク（増資の繰り返し）

```
【メタプラネットのビジネスモデルの構造的問題】
・BTC購入資金 → 株式増資・社債発行で調達
・増資のたびに既存株主の持分が希薄化
・「1株あたりBTC量」が増えなければ株主は損をする

要注意：増資のペース > BTCの価値上昇ペース なら株主は不利
```

### ④ 流動性・ガバナンスリスク

| リスク項目 | 詳細 |
|-----------|------|
| 流動性 | 東証グロース小型株。大量購入・売却時にスプレッドが拡大 |
| ガバナンス | 経営の透明性・日本語開示の質 |
