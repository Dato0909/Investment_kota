# 自民党 政権公約2026「日本列島を、強く豊かに。」
## 強調表現・定量記述 整理文書

- **対象選挙：** 第51回衆議院選挙（2026年1月27日公示・2月8日投開票）
- **スローガン：** 「日本列島を、強く豊かに。」
- **出典：** 政権公約2026（簡易版）・Jファイル2026（総合政策集）
- **作成日：** 2026年3月27日

---

## 1. 経済・財政・賃金

| 記述 | 分類 |
|------|------|
| **名目GDP 1,000兆円**の達成を目指す | 定量 |
| **国民所得 5割以上**アップを目指す（目標：2040年） | 定量・年次 |
| **2030年度までに平均年収を約100万円増加** | 定量・年次 |
| 実質**1%**・名目**3%**の賃金上昇率を継続的に維持 | 定量 |
| 政府全体で**1兆円規模**の支援（賃上げ企業向け成長投資） | 定量 |
| **2040年**に国内投資額年**200兆円**を実現 | 定量・年次 |
| **2030年**までに設備投資の**100%償却**を実現 | 定量・年次 |
| **「100億宣言企業」**による成長投資支援を**抜本的強化** | 強調 |
| 「**責任ある積極財政**」の下、**大胆かつ戦略的な**危機管理投資・成長投資 | 強調 |
| 政府債務残高のGDP比を**着実に低下** | 強調 |
| 「**投資と成長の好循環**」を生み出す | 強調 |
| AI・半導体など**17の戦略分野**に**集中的に**投資 | 定量・強調 |
| 危機管理投資・成長投資の予算 **6.4兆円** | 定量 |
| AI研究開発などにそれぞれ**1,000億〜2,000億円規模**を投入 | 定量 |

---

## 2. 防衛・安全保障

| 記述 | 分類 |
|------|------|
| 防衛費 **2027年度にGDP比2%**を**2年前倒し**で達成 | 定量・年次・強調 |
| 防衛力強化に**1.1兆円**を積み増し | 定量 |
| 2025年度の防衛費 合計約**11兆円** | 定量 |
| 反撃能力等の**強化**・宇宙・サイバー・電磁波領域の体制強化を**加速** | 強調 |
| 安全保障関連3文書を**年内に改定**し防衛装備移転の制限ルールを**撤廃** | 年次・強調 |
| 対外情報機関の**設置** | 強調 |
| 「**外国代理人登録法**」の整備（外国勢力の代理人に届出を**義務付け**） | 強調 |
| レアアース（希土類）など**重要鉱物**の確保を**積極的に**推進 | 強調 |
| 全ての拉致被害者の**即時一括帰国**実現へ**あらゆる手段**を尽くす | 強調 |

---

## 3. エネルギー・脱炭素

| 記述 | 分類 |
|------|------|
| **ガソリン税暫定税率（25.1円/L）**を2025年12月31日に廃止 | 定量・年次 |
| **軽油引取税（17.1円/L）**を2026年4月1日に廃止 | 定量・年次 |
| 1世帯あたり年間**約1.2万円**の減税効果 | 定量 |
| 原子力発電所の**再稼働**を**積極的に**推進 | 強調 |
| 次世代革新炉・フュージョンエネルギーの**早期の**社会実装 | 強調 |
| **エネルギー輸入国からの脱却**を目指す | 強調 |
| 国産ペロブスカイト太陽電池・地熱等を**最大限**活用 | 強調 |

## 4. 少子化・子育て・教育

| 記述 | 分類 |
|------|------|
| **3.6兆円規模**の「こども未来戦略」加速化プランを推進 | 定量 |
| 「こども誰でも通園制度」の**本格実施** | 強調 |
| 標準的な出産費用の自己負担の**無償化** | 強調 |
| **子カルテ普及率を2030年までに100%達成** | 定量・年次 |
| 幼児期から大学院・社会人まで**一気通貫**の公教育の質向上改革を**断行** | 強調 |
| 高校教育改革を「授業料無償化」を機に**強力に進める** | 強調 |
| こども政策を**「強力に推進」**し「こどもまんなか社会」を実現 | 強調 |
| 医療・福祉・介護分野の**確実な賃上げ**を実現 | 強調 |

---

## 5. 税制・生活費

| 記述 | 分類 |
|------|------|
| 飲食料品を**2年間に限り**消費税の対象から除外を**検討加速** | 年次・強調 |
| マイナンバーによる「**公正・公平・迅速**」なプッシュ型給付インフラの構築 | 強調 |
| 中・低所得者（若者・現役世代）の税・社会保険料負担を**軽減** | 強調 |
| 「**給付付き税額控除**」の制度設計を推進 | 強調 |

## 6. 政治改革・憲法

| 記述 | 分類 |
|------|------|
| 衆院議員定数を**1割削減**する法案を**次期国会で成立**を目指す | 定量・年次 |
| 自衛隊明記など**4項目**の憲法改正実現に向け**国民への丁寧な説明を積極的に**展開 | 定量・強調 |
| 政治資金の透明性・公開性を**強化**する第三者機関を**創設** | 強調 |
| 女性議員の割合を現在**12%から2033年までに30%**に引き上げ | 定量・年次 |
| 皇室典範改正を**速やかに**目指す | 強調 |
| インターネット投票・被選挙権年齢の引き下げを**推進** | 強調 |
