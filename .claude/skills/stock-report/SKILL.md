# スキル: stock-report (個別銘柄レポート)

## 概要
指定した銘柄の詳細財務分析レポートを生成します。
PER/PBR・収益性・財務健全性・成長性・バリュートラップ判定を含む総合評価を出力します。

## トリガーパターン (自然言語)
以下のような意図を検出したときにこのスキルを実行してください:
- 「トヨタを分析して」「7203.Tのレポートを出して」
- 「アップルの財務を見せて」「AAPLはどうか？」
- 「この銘柄はバリュートラップ？」「投資してもいい？」
- 「PERとPBRを教えて」「配当はいくら？」
- 「○○と△△を比較して」「どっちがいい銘柄？」
- 「ソニーとパナソニックを比べて」

## 実行方法

### 単一銘柄分析
```bash
python scripts/stock_report.py --ticker <ティッカー>
```

### 複数銘柄比較
```bash
python scripts/stock_report.py --ticker <ティッカー1> <ティッカー2> <ティッカー3>
```

## 主要ティッカー例

### 日本株 (サフィックス .T)
- `7203.T` = トヨタ自動車
- `6758.T` = ソニーグループ
- `9432.T` = NTT
- `8306.T` = 三菱UFJ
- `9984.T` = ソフトバンクグループ
- `6861.T` = キーエンス
- `4502.T` = 武田薬品工業

### 米国株 (サフィックスなし)
- `AAPL` = Apple
- `MSFT` = Microsoft
- `GOOGL` = Alphabet
- `NVDA` = NVIDIA
- `BRK-B` = Berkshire Hathaway
- `JPM` = JPMorgan Chase

### 香港株 (サフィックス .HK)
- `0700.HK` = テンセント
- `9988.HK` = アリババ

## 使用例

```bash
# トヨタ自動車の詳細分析
python scripts/stock_report.py --ticker 7203.T

# AppleとMicrosoftの比較
python scripts/stock_report.py --ticker AAPL MSFT

# 日米3銘柄の比較
python scripts/stock_report.py --ticker 7203.T AAPL 0700.HK
```

## 出力内容

1. **総合評価** — 投資推奨度・スコア・バリュートラップリスク
2. **株価情報** — 現在値・52週高値安値・時価総額
3. **バリュエーション** — PER/PBR/PSR/EV-EBITDA/配当利回り
4. **収益性指標** — ROE/ROA/純利益率/営業利益率
5. **財務健全性** — 流動比率/当座比率/D/E比率
6. **成長性** — 売上成長率/利益成長率
7. **バリュートラップ診断** — レッドフラグ・注意点

## スコアリング基準

| スコア | 評価 |
|-------|------|
| 70以上 | Strong Buy (積極的な買い候補) |
| 55-70 | Buy (買い候補) |
| 40-55 | Hold / Watch (様子見) |
| 25-40 | Weak Hold (弱含み) |
| 25未満 | Avoid (回避推奨) |

## 注意事項
- データはYahoo Finance APIから取得 (24時間キャッシュ)
- 分析は機械的スコアリングであり、定性的要因は含まれません
- 最終的な投資判断はご自身の責任で行ってください
