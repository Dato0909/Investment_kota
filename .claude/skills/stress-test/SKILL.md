# スキル: stress-test (ポートフォリオストレステスト)

## 概要
保有ポートフォリオに対して8つのシナリオでストレステストを実施し、
各シナリオでの推定損失額・損失率を計算します。

## トリガーパターン (自然言語)
以下のような意図を検出したときにこのスキルを実行してください:
- 「ストレステストをして」「リスク分析して」
- 「暴落したらどうなる？」「暴落耐性を見て」
- 「リーマンショック級だったら？」「コロナショックのシミュレーション」
- 「金利が上がったら影響は？」「円高になったらポートフォリオは？」
- 「AIバブル崩壊したら？」「パンデミックが来たら？」
- 「シナリオ一覧を見せて」

## 実行方法

### 全シナリオ実行
```bash
python scripts/stress_test.py
```

### 特定シナリオのみ実行
```bash
python scripts/stress_test.py --scenarios market_crash rate_hike
```

### シナリオ一覧表示
```bash
python scripts/stress_test.py --list-scenarios
```

## 8つのストレステストシナリオ

| キー | シナリオ名 | 市場ショック |
|-----|---------|------------|
| `market_crash` | 市場暴落 (リーマン級) | -40% |
| `rate_hike` | 急激な金利上昇 (+3%) | -20% |
| `yen_appreciation` | 急激な円高 (100円) | -15% |
| `inflation_surge` | インフレ急騰 (8%超) | -18% |
| `geopolitical_risk` | 地政学リスク (紛争) | -20% |
| `tech_bubble_burst` | テックバブル崩壊 | -25% |
| `pandemic` | 新型パンデミック | -30% |
| `mild_recession` | 軽度な景気後退 | -15% |

## 出力内容

1. **シナリオ別サマリー** — 各シナリオの推定損失額・損失率
2. **最悪シナリオ詳細** — 銘柄別の影響額・ショック率
3. **最悪/最良シナリオの特定**

## 注意事項
- ポートフォリオ (`data/portfolio.csv`) が空の場合は実行できません
- 推定値はセクター別の過去データに基づく試算です
- 実際の損失を保証するものではありません
